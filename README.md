# gh-pages demo (React + Tailwind + TypeScript + Zod) - Bun

Quick start:

Install dependencies with Bun and run dev:

```bash
bun install
bun run dev
```

Build and preview:

```bash
bun run build
bun run preview
```

This repo contains a GitHub Actions workflow that builds with Bun and deploys `./dist` to the `gh-pages` branch.
